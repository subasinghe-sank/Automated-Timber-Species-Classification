# Automated Timber Species Classification via Trunk Bark Texture Analysis

## Overview

This project was developed for the **CSC3141 Image Processing Laboratory** at the **University of Peradeniya**.

The system automatically identifies timber species through bark texture analysis using image processing techniques. Image preprocessing, texture feature extraction, and threshold-based classification are used to predict the timber species from bark images.

## Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Tkinter

## Timber Species

* Coconut
* London Plane
* Pine
* Rainbow Eucalyptus

## Features

* Image preprocessing pipeline
* Texture feature extraction
* Threshold-based classification
* GUI-based prediction system
* Visualization of preprocessing stages
