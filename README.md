# Automated Timber Species Classification via Trunk Bark Texture Analysis

## Overview

This project was developed for the CSC3141 Image Processing Laboratory at the University of Peradeniya.

Automated timber species classification through bark texture analysis using image processing techniques.

## Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Tkinter

## Timber Species

* Coconut
* London Plane
* Pine
* Rainbow Eucalyptus

## Features

* Image preprocessing pipeline
* Texture feature extraction
* Threshold-based classification
* GUI-based prediction system
